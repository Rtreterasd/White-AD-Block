Ad Blocker for YouTube

Privacy Policy

Last updated: May 20, 2026

English

Ad Blocker for YouTube is a Chrome extension with one purpose: blocking ads on YouTube websites. The extension works locally in the browser and does not send browsing data to a server.

Data Collection

The extension does not collect, sell, transfer, or share personal data. It does not collect browsing history, messages, form contents, account credentials, payment information, IP address, precise location, or device identifiers.

Local Processing

To block ads on YouTube, the extension may apply Declarative Net Request rules, hide YouTube ad containers and overlays, handle visible YouTube ad skip or close controls, and synchronize the local On/Off state with the YouTube page script.

Local Storage

The extension stores only the On/Off state in chrome.storage.local and a small page state flag on YouTube domains. This data remains local in the user's browser profile.

Remote Code

The extension does not download or execute remote code. All JavaScript, CSS, HTML, and visual assets are packaged with the extension.

Third-Party Links

The Support button may open a third-party donation page only after the user clicks it. The extension does not use that link to transmit browsing data.

Русский

Ad Blocker for YouTube - расширение Chrome с одной целью: блокировать рекламу на сайтах YouTube. Расширение работает локально в браузере и не отправляет данные о просмотре на сервер.

Сбор данных

Расширение не собирает, не продает, не передает и не распространяет персональные данные. Оно не собирает историю браузера, сообщения, содержимое форм, данные аккаунтов, платежную информацию, IP-адрес, точную геолокацию или идентификаторы устройств.

Локальная обработка

Для блокировки рекламы на YouTube расширение может применять правила Declarative Net Request, скрывать рекламные контейнеры и оверлеи YouTube, обрабатывать видимые кнопки пропуска или закрытия рекламы и синхронизировать локальное состояние On/Off со скриптом страницы YouTube.

Локальное хранение

Расширение хранит только состояние On/Off в chrome.storage.local и небольшой флаг состояния страницы на доменах YouTube. Эти данные остаются локально в профиле браузера пользователя.

Удаленный код

Расширение не загружает и не выполняет удаленный код. Все JavaScript, CSS, HTML и визуальные файлы поставляются внутри расширения.

Сторонние ссылки

Кнопка Support может открыть стороннюю страницу пожертвований только после клика пользователя. Расширение не использует эту ссылку для передачи данных о просмотре.

Contact
